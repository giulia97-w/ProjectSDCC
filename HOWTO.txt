Per l'installazione dell'applicazione all'interno della cartella scaricata da github contenente i 4 file(ProjectSDCC.py, HOWTO.txt, covidCell.jpg e RelazioneProjectSDCC)  
lanciare da terminale ProjectSDCC.py 
o aprire il file .py

Se non presenti installare le libreria Python utilizzate per l'implementazione della GUI

- Python Requests
- Python Pillow
- Python Tkinter

tramite i comandi:

pip install requests 

pip install pillow

In Python Tkinter è una libreria di default, durante l'installazione di Python è necessario 
spuntare l'installazione di Tcl/Tk.

Se non presente può essere installate con il comando:

pip install tk
